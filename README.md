# 📊 Sales Analysis Project

![](sales.png)

## 📌 Overview
This project focuses on analyzing sales data to uncover meaningful insights, identify trends, and support data-driven decision-making. The analysis helps understand business performance across different dimensions such as time, product categories, and customer segments.

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-performing products and categories
- Understand monthly and yearly sales trends
- Evaluate the impact of discounts and offers
- Discover key insights to improve revenue

## 📂 Dataset Description
The dataset contains the following key fields:
- **Order ID** – Unique identifier for each order
- **Order Date** – Date of purchase
- **Product Name** – Name of the product sold
- **Category / Sub-category** – Product classification
- **Quantity** – Number of units sold
- **Sales** – Revenue generated
- **Discount** – Discount applied
- **Profit** – Profit earned after discount

## 🔍 Key Analysis Performed
- Time-based analysis (Monthly / Yearly trends)
- Category-wise and product-wise sales comparison
- Top N products based on revenue
- Sales vs Profit analysis
- Impact of discounts on sales performance

## 📈 Tools & Technologies Used
- Python (Pandas, NumPy, Matplotlib)
- Power BI / Excel (for dashboards)
- DAX (for calculated measures)

## 📊 Insights & Findings
- Identified high-performing product categories
- Observed seasonal trends in sales
- Found correlation between discount and sales volume
- Highlighted areas of low profitability

## 🚀 Conclusion
The analysis provides actionable insights that can help optimize pricing strategies, improve product focus, and enhance overall business performance.
